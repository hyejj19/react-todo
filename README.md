## 😸 정말 간단한 투두리스트 version_1.1

---

### 💜 소개

리액트를 사용해 개발한 정말 간단한 투두리스트 입니다.
version_1.1에서는 json-server를 사용하여 fetch를 활용했습니다.

### 💜 사용한 기술

- React.js
- styled-components
- json-server

### 💜 주요 기능

- Create : 할 일 생성
- Read : 할 일 목록 조회
- Update : inline-edit 방식으로 할 일 수정
- Delete : 할 일 삭제
