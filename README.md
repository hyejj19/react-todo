## 😸 정말 간단한 투두리스트 version_1.0

---

### 💜 소개

리액트를 사용해 개발한 정말 간단한 투두리스트 입니다.

version_1.0에서는 redux-toolkit을 활용해 상태를 관리하고, localStorage에 저장하여 관리했습니다.

### 💜 배포 링크

- [배포 링크](https://hyejj19.github.io/react-todo/)

### 💜 사용한 기술

- React.js
- styled-components
- redux-toolkit

### 💜 주요 기능

- Create : 할 일 생성
- Read : 할 일 목록 조회
- Delete : 할 일 삭제
